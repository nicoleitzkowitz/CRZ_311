# CRZ_311

NYC congestion relief zone noise complaint DiD analyses <br>
Created 2026 JAN 14 by Nicole Itzkowitz <br>

code <br>
- CRZ_311_noise_complaints_2026JAN14.rmd - data processing and analysis 
- 2026_ICBEN_slides_2026_JUN11.r - create visuals for ICBEN slides

 data <br>
- crz_control/ - shapefile boundary for control area, created in QGIS
- MTA Central Business District Geofence_ Beginning June 2024_20250619/ - shapefile boundary for CRZ, from MTA
- crz_ctrl_noise_complaints.gpkg - shapefile of geocoded noise complaints in CRZ and CTRL 
- note: may need to download 311 call data from NYCOpenData directly, as R package isn't that good for large files and the downloaded csv is too pig to push to github
